# Progress Tracker

Progress Tracker is a small python (3.11) project, that runs with a basic installation of python.
It runs anywhere where you are allowed to execute python scripts.  
The script starts up a window where the user can maneuver around to enter his data, view and edit it,
which is saved to the corresponding `data.sqlite` file - which is always in the same folder as the script.

## Features

The application allows for creating of own categories. Every category can then
be edited and viewed.  
Categories can be amended, meaning columns can be deleted or
added.   
Every table needs at least one primary key, this will be checked when the user tries to create a category.
